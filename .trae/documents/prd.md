## 1. 产品概述

每日热点汇聚是一个现代化的新闻聚合平台，自动抓取并展示来自16个主流平台的24小时内最新热点资讯。
- 解决用户需要频繁切换多个平台获取信息的痛点，提供一站式热点浏览体验
- 面向追求高效信息获取的互联网用户，打造简洁优雅的阅读体验

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览所有平台热点、搜索筛选、收藏本地存储 |

### 2.2 功能模块
1. **首页**: 平台导航、热点瀑布流、实时更新指示器、搜索筛选
2. **详情页**: 新闻详情预览、原文跳转、分享功能

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 平台导航栏 | 16个平台图标横向滚动，点击筛选对应平台热点 |
| 首页 | 热点卡片流 | 瀑布流布局展示热点标题、来源、时间、热度指标 |
| 首页 | 搜索栏 | 支持关键词搜索、时间筛选、热度排序 |
| 首页 | 实时更新 | 自动轮询更新，显示最新热点数量提示 |
| 详情页 | 内容预览 | 展示热点摘要、配图、元信息 |
| 详情页 | 操作栏 | 收藏、分享、跳转原文按钮 |

## 3. 核心流程

用户打开应用 → 自动加载各平台最新热点 → 浏览瀑布流内容 → 点击平台标签筛选 → 点击卡片查看详情 → 跳转原文阅读完整内容

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**: 深色主题 (#0a0a0f 背景) + 渐变强调色 (琥珀金 #f59e0b 到橙红 #ef4444)
- **辅助色**: 平台专属色彩标识（B站粉、微博橙、知乎蓝等）
- **按钮风格**: 圆角胶囊按钮，微光渐变边框，悬停发光效果
- **字体**: 标题使用 "Noto Sans SC" 粗体，正文使用 "Inter" 优化中文显示
- **布局风格**: 响应式瀑布流 + 固定顶部导航 + 侧边快捷操作
- **图标风格**: 各平台品牌图标 + 线性功能图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 顶部导航 | 毛玻璃效果背景、Logo居左、搜索框居中、时间显示居右 |
| 首页 | 平台标签栏 | 横向滚动、平台图标+名称、选中态渐变下划线 |
| 首页 | 热点卡片 | 圆角卡片、平台色左边框、标题+摘要+元信息、悬停微动效 |
| 首页 | 侧边栏 | 快捷日期筛选、热度排行TOP5、收藏入口 |
| 详情页 | 内容区 | 居中卡片、大标题、配图、摘要、元信息标签 |
| 详情页 | 操作按钮 | 底部固定、收藏/分享/跳转三个胶囊按钮 |

### 4.3 响应式设计
- **桌面优先**: 3列瀑布流布局，侧边栏常驻
- **平板适配**: 2列瀑布流，侧边栏折叠为底部抽屉
- **移动端**: 单列卡片流，平台导航变为底部Tab，搜索变为顶部展开

### 4.4 动效设计
- **页面加载**: 骨架屏渐显 → 卡片依次淡入上滑（stagger动画）
- **卡片悬停**: 轻微上浮 + 阴影加深 + 平台色边框发光
- **平台切换**: 标签滑动指示器 + 内容区淡入淡出
- **新热点提示**: 顶部滑入Toast通知，点击刷新
